חיבור ל-Google Sheets
הגדרה חד-פעמית — לאחר מכן הכלי יכתוב ישירות לגיליון שלך
שלב 1 — צור פרויקט ב-Google Cloud
  1. פתח: console.cloud.google.com
  2. לחץ "Select a project" → "New Project"
  3. תן שם כלשהו → "Create"
  4. תפריט שמאל → "APIs & Services" → "Library"
  5. חפש "Google Sheets API" → Enable
  6. תפריט שמאל → "APIs & Services" → "OAuth consent screen"
  7. בחר "External" → Fill App name → Save
  8. תפריט שמאל → "Credentials" → "Create Credentials" → "OAuth client ID"
  9. Application type: Web application
  10. Authorized JavaScript origins: http://localhost + כתובת הקובץ שלך
  11. לחץ "Create" — קבל Client ID ו-Client Secret
